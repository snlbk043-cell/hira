RCPL Kurnool Campa Cola – Smart Dynamic HIRA Generator

Enter activity and sub-activity for up to 30 tasks. The tool analyses equipment, chemical/process condition, energy source, job mode, and likely failure event to generate practical HIRA rows for a soft beverage / CSD plant.

How this smart version works

This is not simple keyword filling. It checks the task as an event pathway: activity + sub-activity + area/equipment + job mode + site condition. Example: “CIP chemical sample collection from black knob” generates hazards for pressurised residual caustic/acid, hose jerk, splash trajectory, pH verification, closed sampling container with tap valve, emergency shower, face shield. Unknown tasks are still analysed using hazard-energy logic.
Use this as a strong draft for RCPL implementation. Final HIRA must be reviewed and approved by EHS, area owner and maintenance/production before release.
